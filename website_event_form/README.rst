This module was written to extend the functionality of the *website_event*
module by adding a snippet form that allows users to create events from a form
fill out.

**Table of contents**

.. contents::
   :local:

Configuration
=============

Once installed, Go to Odoo in website editor mode, and drag-drop
"Events Snippet" form to your designed page and section for users to fill and
create events on the backend by submitting the form.

Usage
=====

To use this module, you need to:

Drag Drop The Event Form snippet from Website Edit.
